# EduFlow App

Next.js 15 production app — school management SaaS.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Stack
- Next.js 15 (App Router, Turbopack)
- shadcn/ui PRO V6 design system (zinc base, oklch)
- Tailwind CSS v4
- Lucide React icons
- Recharts
